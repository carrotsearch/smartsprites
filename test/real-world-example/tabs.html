Carrot2 Search Results Clustering Engine

Carrot2 is an Open Source Search Results Clustering Engine. It can automatically organize (cluster) search results into thematic categories.

Choose where to search:

  • Web Carrot2 will search the Web with www.etools.ch and automatically organize the results into thematic categories
  • News Carrot2 will send the query to Yahoo! News and automatically organize the results into thematic categories
  • Yahoo! Carrot2 will search the Web with Yahoo! and automatically organize the results into thematic categories
  • Google Carrot2 will search the Web with Google and automatically organize the results into thematic categories
  • Jobs Carrot2 will search for jobs and automatically organize the results into thematic categories
